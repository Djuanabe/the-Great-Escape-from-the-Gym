# チャージランナー (Charge Runner)

ワンボタンの横スクロール・アクションゲーム。バスケットボールが自動で前進し、跳ね続けます。
チャージ・色切り替え・すり抜けを駆使して、ランダム生成される無限のコートをどこまでも進もう。

## 遊び方

操作は**ボタン1つ**（スペースキー / クリック / タップ）。

- **地上付近で押し続ける** … 減速しながらチャージ
- **離す** … チャージ分だけ加速してジャンプ（チャージが大きいほど高度は下がる／上限で自動解放）
- **空中でタップ** … ボールの色を切り替え（オレンジ ⇔ 黒）

### 妨害要素

- **穴** … 跳び越える。広い穴の中央には色付きの「救済の浮島」が出ることがある（自分と違う色なら足場になる／同じ色だとすり抜けて落下）
- **壁・移動壁** …
  - **色付き（オレンジ/黒）** … 自分の色と合わせるとすり抜けられる。跳び越えてもOK
  - **白** … どの色でもすり抜け不可。跳んで避けるしかない
- 進むほど色付きの壁の割合が増え、スピードも等加速度的に少しずつ上がっていく

## 起動方法

静的なHTMLファイル1枚だけで動きます。

- `index.html` をブラウザで直接開く、または
- 簡易サーバーで配信：
  ```bash
  npx serve .
  # もしくは
  python -m http.server
  ```

## 技術メモ

- 依存ライブラリなしの単一ファイル（HTML5 Canvas + Web Audio）
- ステージはシード付き乱数で再現可能に自動生成。半画面ごとに「予測軌道」を計算し、それを避ける形で妨害要素を配置（猶予は進むほど狭まる）
- 効果音は Web Audio API で生成（チャージ音は溜まるほど音階が上昇）
